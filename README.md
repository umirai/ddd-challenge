## Startup

```bash
$ docker compose up -d
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Migration

```bash
# prisma setup
$ yarn prisma:generate

# db-test: migrate & seeding
$ yarn migrate:test

# db: reset
$ yarn migrate:reset

# db: reset & seeding
$ yarn migrate:init

# db: migrate & seeding
$ yarn migrate:dev
```

## Test

```bash
# unit tests & db-reset
$ yarn test:reset

# unit tests
$ yarn test
```
